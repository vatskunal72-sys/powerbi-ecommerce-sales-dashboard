# powerbi-ecommerce-sales-dashboard
Interactive Power BI dashboard analyzing ecommerce sales, profit trends, customer insights, and payment methods.

# 🛒 Ecommerce Sales Dashboard

## 📌 Objective

Analyze ecommerce sales data to understand profit trends, customer behavior, state-wise sales, and payment methods.

---

# 🛠 Tools Used

* Power BI
* Excel

---

# 📊 Dashboard Features

* Sales & Profit Overview
* State-wise Sales Analysis
* Category-wise Quantity Analysis
* Monthly Profit Trends
* Customer-wise Sales Insights
* Payment Method Analysis
* Sub-Category Profit Comparison

---

# 📈 Key Insights

* Maharashtra generated the highest sales
* Clothing category contributed maximum quantity
* COD was the most used payment method
* Printers generated highest profit among sub-categories
* Profit varied significantly across months
